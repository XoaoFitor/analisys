# Análise de Intenção de Compra Online – EBAC

Aplicação interativa desenvolvida em **Python** e **Streamlit** para análise exploratória do dataset **Online Shoppers Intention**, do UCI Machine Learning Repository.  
O objetivo é visualizar o comportamento dos usuários em um e-commerce e identificar padrões relacionados à variável-alvo **Revenue** (intenção de compra).

---

##  Funcionalidades

- **Upload de CSV** ou uso do arquivo padrão (`online_shoppers_intention.csv`)
- **Visualização interativa** dos dados brutos e filtrados
- **Filtros dinâmicos** por mês, tipo de visitante e fim de semana
- **Escolha da variável categórica de análise**
- **Gráficos comparativos** lado a lado:
  -  Barras horizontais
  -  Gráfico de pizza
- **Resumo de métricas**
  - Contagem por categoria
  - Soma de métricas numéricas selecionadas
- **Exportação para Excel (.xlsx)** com múltiplas abas:
  - Dados brutos  
  - Dados filtrados  
  - Resumo por categoria  
  - Contagens ou somas agrupadas
